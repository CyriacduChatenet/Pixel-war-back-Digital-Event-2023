# Pixel War Back-end

## Installation

```bash
$ git clone https://github.com/CyriacduChatenet/Pixel-war-back-Digital-Event-2023.git
$ cd Pixel-war-back-Digital-Event-2023
$ npm install
```

## Running the app

```bash
# development
$ make up
$ npm run start:dev
```
## Usage

Documentation is available at [http://localhost:3000/api](http://localhost:3000/api)
